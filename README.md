# Lab-11-14-16-17

## Задание к 11 лабораторной
1. Заполнение массива с консоли
2. Преобразование массива
3. Печать массива
4. Найти сумму элементов массива, за исключением минимального и максимального значения.

## Задание к 14 лабораторной
1.Заполнение массива значениями функции f(x) = 5x³ + 2x² + 15x - 6
2.Печать массива
3.Вычисление суммы элементов в заданном диапазоне индексов
4.Поиск элемента в массиве
5.Добавление последнего элемента ко всем элементам массива

## Задание к 16 лабораторной
1.Заполнение массива значениями функции
2.Поиск минимального элемента в массиве
3.Вставка k элементов со значением -999 после минимального элемента
4.Печать массива до и после вставки

## Задание к 17 лабораторной 
Реализация функций сортировки массива:
1.пузырьковой сортировки
2.шейкерной (коктейльной) сортировки
3.сортировки выбором
4.сортировки вставками
5. Провести эксперимент, сравнивающий скорость выполнения каждой из сортировок

## Реализованные функции
1. void print_double(double* a, int n)
Назначение: Печать массива вещественных чисел
Параметры:
a - указатель на массив double
n - количество элементов для вывода
Формат вывода: числа с двумя знаками после запятой, разделенные пробелами
Возвращает: ничего

2. void print_int(int* a, int n)
Назначение: Печать массива целых чисел
Параметры:
a - указатель на массив int
n - количество элементов для вывода
Особенность: выводит только первые 20 элементов для больших массивов
Возвращает: ничего

3. void prog1()
Назначение: Вычисление суммы всех элементов, кроме минимального и максимального
Алгоритм:
Ввод размера массива (ограничение: 1-10 элементов)
Ввод элементов массива
Поиск минимального и максимального значений
Вычисление суммы элементов, отличных от min и max
Возвращает: ничего (результат выводится на экран)

4. void fill_double(double* a, int n)
Назначение: Заполнение массива значениями функции f(x) = 5x³ + 2x² + 15x - 6
Параметры:
a - указатель на массив double
n - количество элементов
Логика: пользователь задает начальное x и шаг dx, функция вычисляется для x, x+dx, x+2dx, ...
Возвращает: ничего

5. double sum_range(double* a, int b, int e)
Назначение: Вычисление суммы элементов массива в заданном диапазоне индексов
Параметры:
a - указатель на массив double
b - начальный индекс (включительно)
e - конечный индекс (включительно)
Возвращает: сумму элементов a[b] + a[b+1] + ... + a[e]

6. int find_val(double* a, int n, double v)
Назначение: Поиск элемента в массиве
Параметры:
a - указатель на массив double
n - количество элементов
v - искомое значение
Точность: сравнение с допуском 0.001
Возвращает: индекс первого найденного элемента или -1, если не найден

7. void add_last(double* a, int n)
Назначение: Добавление последнего элемента массива ко всем элементам
Параметры:
a - указатель на массив double
n - количество элементов
Операция: a[i] = a[i] + a[n-1] для всех i от 0 до n-1
Возвращает: ничего

8. void prog2()
Назначение: Работа с массивом вещественных чисел
Параметры: нет
Возвращает: ничего
Реализованные операции:
Заполнение массива значениями функции
Вычисление суммы в указанном диапазоне
Поиск элемента по значению
Добавление последнего элемента ко всем элементам

9. int find_min_idx(double* a, int n)
Назначение: Поиск индекса минимального элемента в массиве
Параметры:
a - указатель на массив double
n - количество элементов
Возвращает: индекс первого минимального элемента

10. void insert_after_min(double* a, int* n, int k)
Назначение: Вставка k элементов со значением -999 после минимального элемента
Параметры:
a - указатель на массив double
n - указатель на количество элементов (изменяется)
k - количество элементов для вставки
Логика:
Находит индекс минимального элемента
Сдвигает элементы справа от минимального на k позиций
Вставляет k элементов -999.0
Увеличивает размер массива на k
Возвращает: ничего
